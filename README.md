# Support Tickets

## Sobre

<p>This is a simple support ticket project to explore basic concepts about APIs using pure javascript without using frameworks.</p>

## Technologies

- `Javascript`
- `Node.js`
- `Routes`
- `Middlewares`
- `Controllers`

## Installation

```bash
# clone project
$ git clone https://github.com/CiceroEduardo84/Support_tickets_API.git

# run interface
$ npm run dev
```
